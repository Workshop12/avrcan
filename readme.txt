Using Gerald's ATMEGA64M1 Board
===============================


Don't Install board support:
    https://thomasonw.github.io/ATmegaxxM1-C1/package_thomasonw_ATmegaxxM1-C1_index.json

Do install board support:
    https://geraldjust.github.io/ATmegaxxM1-C1/package_thomasonw_ATmegaxxM1-C1_index.json


Install can var library
    git clone https://github.com/thomasonw/avr_can.git
